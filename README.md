# Global Superstore Sales Analysis

# Project Overview
This project analyzes sales performance using the Global Superstore dataset. The goal was to reveal business insights related to revenue trends, regional performance and product profitability using an interactive Power BI dashboard.

# Tools Used
Power BI
Data Cleaning and Transformation(Power Query)
DAX for calculations

# Key Metrics
Total Sales
Total Profit
Number of Orders
Profit Margin

# Dashboard Features
Sales performace by region
Profiy by product category
Monthly sales trend analysis
Interactive filters for year and date range

# Key Insights
The highest sales revenue were generated by the Central and South regions, this indicate strong market demands in these regions.
Sales show a steady rise trend from 2011-2014, and this suggests consistent business growth.
Technology products contribute the largest share of profit, which makes them the most profitable category.
